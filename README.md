## ADJoin
* 설정 파일 정보
  * Group.txt : Local Administrators 그룹에 넣을 Group 정보 
  * UserInfo.txt : AD Join 시 사용할 Account Info

* 사용 방법
  ``` powershell
    DomainJoin.ps1 -strDomainNM [Join할 Domain Name] -strUser [Domain할 때 사용하는 Account Info]
  ```
